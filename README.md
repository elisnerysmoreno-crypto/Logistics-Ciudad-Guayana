# Logistics - Planificador de Entregas (Ciudad Guayana)

Sistema inteligente de optimización logística desarrollado en Java, diseñado para gestionar rutas de entrega eficientes en el estado Bolívar, Venezuela.

## 🚀 Características Principales
* **Optimización con Dijkstra:** Implementación del algoritmo de grafos para hallar la ruta más corta entre puntos de entrega.
* **Arquitectura MVC:** Separación clara entre el Modelo (datos), la Vista (interfaz Swing) y el Controlador (lógica).
* **Simulación en Tiempo Real:** Visualización animada del recorrido del vehículo sobre el mapa.
* **Gestión de Carga:** Control de peso y volumen de paquetes por cada punto de entrega.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java 17+
* **Interfaz Gráfica:** Java Swing / NetBeans GUI Builder
* **Estructuras de Datos:** Grafos (Listas de adyacencia), Nodos y Aristas pesadas.
* **Persistencia:** Manejo de archivos planos (.txt) para importación de datos y generación de recibos.

## 📋 Requerimientos del Sistema
1. Tener instalada la **Java Virtual Machine (JVM)**.
2. Para desarrollo: **NetBeans IDE** (el proyecto incluye los archivos `.nbproject`).
